Lists

Items in an ordered list will be represented as:

  1. First item in the ordered list
  2. Second item in the ordered list
  3. Third item in the ordered list

Items in an unordered list will be represented as:

  • First item in the unordered list
  • Second item in the unordered list
  • Third item in the unordered list

Items in an nested list can be represented as:

  1. ordered list item
    • unordered list item nested under ordered list item
  2. second ordered list item
    • unordered list item nested under ordered list item
